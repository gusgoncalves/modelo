Sistema Modelo para base da PMPG
=======
# CodeIgniter 4

## Para iniciar o processo
Prompt de Comando, navegue para o local desejado para a instalação, preferencialmente no seu servidor local http ou www .

faça o clone do repositório no VSCode com o comando:
```bash
git clone https://github.com/gusgoncalves/modelo.git
```
Navegue para a pasta criada
```bash
cd modelo
```
```bash
code .
```
Abrindo o VSCode execute o terminal dentro do próprio VSCode
## Setup 
```php
php spark serve
```
abra o navegador e siga os passos da instalção ou execute pelo navegador:
[http://localhost:8080](http://localhost:8080)

## Banco de Dados

Mysql (MariaDB) ou Postgres
É necessário um banco de dados criado, usuário e senha para configuração
