Sistema Modelo para base da PMPG
=======
# CodeIgniter 4

## Para iniciar o processo

faça o clone do repositório no VSCode
```git
    git clone https://github.com/gusgoncalves/modelo.git
```
## Banco de Dados

Mysql (MariaDB) ou Postgres
É necessário um banco de dados criado, usuário e senha para configuração

## Setup

No VSCode faça 
```php
  php spark serve
```
abra o navegador e siga os passos da instalção
Ou execute pelo navegador:
[http://localhost:8080](http://localhost:8080)
